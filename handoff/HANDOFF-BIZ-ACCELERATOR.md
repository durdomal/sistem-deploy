### 1.2 Готовые связки под первую прибыль

**Связка A — «Sell Sites Local Business» (~$7 600/мес на клиента):**
```
Google Maps scan (n8n workflow #2) → marketing-audit → site-landing → sales-funnel закрытие
```
Все компоненты есть. Первый чек 24–72 часа реалистично.

**Связка B — «Retainer Agency» (~$16 500/мес × ретейнер):**
```
niche-comment-miner (боли) → lead-outreach (холодный аутрич) → marketing-audit (бесплатный аудит-крючок) → retainer-agency (продажа)
```
Первый ретейнер 2–4 недели реалистично.

**Связка C — «Booking Agent для стоматологий Барселоны» (пилот):**
```
lead-outreach → booking-agent демо → пилот 1 месяц → продажа модели
```
Не мгновенная прибыль, но пилот запустить можно за неделю.

**Связка D — ITV Costa Blanca / универсальный подбор:**
```
itv-deal-card (или unit-economics для не-авто) → сделка → маржа
```
Работает уже сейчас, не требует ничего кроме входящего запроса.

## 3. Что нужно от акселератор-чата (вопросы к решению)

Sistem-чат сделал инфраструктуру. **Акселератор должен решить бизнес-стратегию**:

1. **Первая ниша для sell-sites-localbiz:** стоматологии Costa Blanca / автосервисы / рестораны / фитнес? — от этого зависит первый выхлоп через n8n workflow.
2. **Цена первого лендинга:** €800–1200 разовый или €300/мес? (в sistem-power-pack v0.3 модель ~$7600/мес — это уже с ретейнером на поддержку).
3. **Первая ниша для retainer-agency:** та же или другая? Диверсификация или фокус?
4. **Пилот booking-agent:** делаем сейчас (открытие рынка) или после первых денег с A/B?
5. **Бюджет на APIs первого месяца:** Apify ~€49/мес, Firecrawl free 1k/мес хватит, Meta Ads MCP — €0 (только OAuth). Итого ~€49–100/мес.
